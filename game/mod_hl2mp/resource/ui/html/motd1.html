Gabe Mod+
Sandbox Mod inspired by JBMod + GMod
Welcome
Gabe Mod is an AI-coded sandbox mod inspired by JBMod.
Build and explore throughout the mod!
Newest Features - Version Eight
  • Added updated Physgun, ported from Half-Life 2 Sandbox and Adnan’s rotational Gravity Gun. Supports rotation with the Use key.
  • New multitool (mode select in spawn menu, explode, duplicator, decal, renderfx, FACEPOSER (sort of)).
  • Lua support. do lua
  • Dynamic spawn menu with keyvalues support.
  • Singleplayer mode (smoother physics, effects work properly).
  • Sticky launcher (re-added).
  • Enemy launcher.
  • New watermark.
  • New game dialog has map icons that are 128x256 and go in materials/vgui/maps/mapname.vmt and vtf
  • New console commands.
  • Bug fixes, fixed bug with hl2 maps crashing by adding a stub autosave ent.
  • HTML tutorials menu (BETA).
  • Half-Life 2 campaign maps work better.
  • Changelog implemented as a GameUI panel.
  • NPC ragdoll physics improvements.
  • Bunnyhopping support.
  • Credits added to main menu.
  • And much more.
Changes
  • Switch between per-material and team footsteps with gabeplus_permaterialfootsteps 1/0
  • Added trees and sand to pond map
  • The map Sandbox will no longer be included in Gabe Mod
Known Bugs
  • Certain effects do not display in multiplayer, singleplayer works fine
  • trigger_changelevel spawns you far into the map, not at the start
  • Saving crashes the game
Notes
  • This mod is under active development
  • Bugs and crashes are somewhat expected
  • Commands use the prefixes gabe+_,gabeplus_, orgabe_
